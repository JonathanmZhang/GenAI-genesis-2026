# AlignAI – Desk Stretch Coach

AlignAI is an Expo + React Native app that guides you through desk‑friendly stretches and gives instant AI‑powered feedback on your form.

## How it works

- The user selects soreness levels for the neck, shoulders, and lower back via sliders.
- The app guides the user with instructions plus a camera preview and framing box.
- After each session, a generative AI coach:
  - Computes a form score from 0–100.
  - Generates a short session summary.
  - Returns a set of concrete tips tailored to the specific stretch.

## Installation and running

From the `alignAI` project folder:

```bash
npm install
npx expo start
```

Then either:

Scan the QR code in the terminal/Expo Dev Tools with the Expo Go app, or

Press the shortcut in the Expo CLI to run on an iOS or Android simulator.
